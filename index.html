このサイトについて

目的

このwebサイトは私の卒業研究である、「オンライン授業等で用いる質問受付システム『StuMee』の開発」 の進捗を管理するためにGitHub Pagesを用いて作成しました。

主に進捗報告及び、参考リンクなどをまとめていく予定です。

GitHub Pagesとは

GitHub Pagesとは、GitHubに専用のリポジトリを作成し、静的なファイルを置くことで ホームページやブログなどを簡単に作成できるサービスです。

Gitの知識さえあれば、基本無料でサーバーの設定等の手間もいらないので、 WordPress等データベースを用いない場合はおすすめです。
